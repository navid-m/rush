# Rush

Rush is a Git repository visualization tool.

## Requirements to build

- [Bun](https://bun.com)
- [The Go Compiler](https://go.dev)
- [GNU Make](https://www.gnu.org/software/make/manual/make.html)

## Build instructions

Run:

```
make all
```
